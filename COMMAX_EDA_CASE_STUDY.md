# Commax 출하 데이터 EDA 사례

## 분석 질문

출하 데이터의 품질과 수요 구조를 확인하고, 품목·분류·시점별 집중도와 계절성을 파악해 이후 예측 모델링에서 우선 검증할 가설을 정리했습니다.

## 데이터 스냅샷

- 20,096행 × 240열, 분석 기간은 2015-01부터 2025-08까지입니다.
- 품목은 157개이며, 타깃 `value`의 31.06%가 0입니다.
- 외부 경제지표 85개 중 결측값이 있는 지표가 67개입니다.
- 원본 파일은 저장소에 포함하지 않으며, 포트폴리오 공개 사용 권한은 확인된 상태입니다. 자세한 계보는 [DATA_CARD.md](DATA_CARD.md)를 참고하세요.

## 핵심 발견과 해석

| 발견 | 해석 | 모델링/업무상 다음 조치 |
| --- | --- | --- |
| `value`의 31.06%가 0이고 오른쪽 꼬리가 매우 김 | 평균 기반 모델은 희소 수요와 대량 출하를 동시에 잘 설명하기 어렵다 | ADI/CV²로 수요 패턴을 재검증하고 intermittent-demand 기준선(Croston 계열 등)을 분리 비교 |
| 상위 20개 품목이 누적 출하량의 53.2%를 차지 | SKU별 모델 투자 우선순위가 다르다 | ABC 분류 후 상위 SKU와 롱테일 SKU의 모델·평가지표를 분리 |
| Door Entry 분류가 전체 출하량의 66.1% | 특정 분류의 변화가 전체 추세를 지배할 수 있다 | 전체 집계 모델만 보지 말고 분류·품목 수준 계층 예측을 검증 |
| 12월 출하가 높고 월별 분해에서 계절성 진폭이 큼 | 연간 계절성이 존재할 가능성이 있다 | rolling backtest에서 연말 구간을 반드시 포함하고, 예측 horizon별 성능을 비교 |
| 2025년은 8개월만 포함 | 연간 합계를 전년과 단순 비교하면 하락폭을 과장한다 | 동일 월 기준 YTD 비교로 정규화하고 관측 종료일을 대시보드에 표시 |
| 외부 지표에 결측이 많음 | 무분별한 외생 변수 투입은 누수·결측 대치 편향을 만든다 | 변수별 발표 시점, 결측 패턴, lag를 정의한 뒤 ablation으로 효과 검증 |

## 비판적으로 봐야 할 부분

- 노트북의 IQR 이상값 비율은 **오류가 아니라 수요 분포의 특성일 수 있습니다**. 제거 전에 프로모션·대형 프로젝트·기록 오류를 구분해야 합니다.
- `value=0`이 실제 무수요인지, 품절인지, 미집계인지가 확인되지 않았습니다. 이 라벨의 의미가 바뀌면 모델과 KPI가 모두 달라집니다.
- 연도별 총량 추세는 품목 구성 변화와 신규/단종 SKU 영향을 섞습니다. 같은 SKU 코호트 기준의 비교가 추가로 필요합니다.
- 외부 지표는 미래 시점에 알 수 있었는지(발표 지연 포함)를 검증하지 않아, 현재 단계에서는 예측 변수로 사용하면 안 됩니다.

## 포트폴리오에서 보여 줄 결론

이 사례의 강점은 ‘Prophet을 적용했다’가 아니라, 희소 수요·집중도·계절성·외생 변수의 시점 정합성이라는 **모델링 전제조건을 데이터로 검증했다**는 데 있습니다. 다음 단계는 이 가설을 rolling backtest와 기준선 비교로 반증하는 것입니다.
